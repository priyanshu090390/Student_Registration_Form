# Student_Registration_Form
This form lets you enter details of student(user) and save it for future purpose. If in case some wrong entries are saved or you want to update any entry afterwards you can do the same by the option provided on the saved list of registered students.

## Description
A responsive Student Registration System built using HTML, CSS, and JavaScript.  
It allows users to add, edit, delete, and store student records using localStorage.

## Features
- Add / Edit / Delete student records
- Data persistence using Local Storage
- Input validation
- Fully responsive (Mobile, Tablet, Desktop)
- Dynamic vertical scrollbar
- Clean UI & UX

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
